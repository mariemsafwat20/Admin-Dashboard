To install dependencies: npm i,
To run the project: ng serve --open

The project is built using Angular v.20 and Angular Material.
 
The Orders API is not available, so products data is used instead.

Project structure:
The core folder contains shared pipes, models, and services.
The layouts folder contains the base layouts for the application, such as admin or user layouts.
The components folder contains all components that render the application pages.
